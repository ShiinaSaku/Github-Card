üöÄ Usage

Generate a GitHub profile card by using the following URL format:

https://card.shiina.xyz/card/:username
Example:
https://card.shiina.xyz/card/ShiinaSaku

Use this in your README or website to display dynamic GitHub profile cards!

‚öôÔ∏è Parameters

All parameters are optional. Customize your card with the following options:

Parameter Type Description
theme string Theme name (see themes section)
title_color string Hex color without #
text_color string Hex color without #
icon_color string Hex color without #
bg_color string Hex color without #
border_color string Hex color without #
hide_border boolean true to hide border
compact boolean false to show bio and labels
fields string Comma-separated list: languages or all
Custom colors example:
https://card.shiina.xyz/card/ShiinaSaku?bg_color=0d1117&text_color=c9d1d9&title_color=58a6ff

üìä What it shows

The card displays the following information:

  • Total stars across owned repositories (excluding forks)
  • Total commits for the current year
  • Total issues (open + closed)
  • Total owned repositories (excluding forks)
  • Total pull requests (open + closed + merged)
  • Top 5 programming languages
